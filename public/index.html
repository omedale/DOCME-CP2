    DocMe API DOCUMENTATION

    Introduction

    Welcome to the DocME API! Our APIs provide you all you need to develop a documement management system, you can create document, complete with roles and privileges.

    Each document defines access rights; the document defines which roles can access it. Also, each document specifies the date it was created. All users are categorize by roles and it is the responisbility of the admin to change role at any point in time.

    All APIs in DocMe are RESTful, you can be assure of consistent response for all request made.

    Authentication

    All routes were protected with JSON WEB TOKEN which is send to client after successful registration or login

    import jwt from ('jswonwebtoken');

const token = jwt.sign('userInfo');

    Always set token as Authorization in the header before making any request.

    An authorization token is required to access all routes except Login and Register routes. Try the register or Login route(for an existing users) to get a token.

    A token generated expires 24 hours after it was generated.
    Kittn uses API keys to allow access to the API. You can register a new Kittn API key at our developer portal.

    The header should be set in the following format:

    Authorization: token

    You must replace token with the one you get after authentication.

    Errors

    This error section is stored in a separate file in `includes/_errors.md`. Slate allows you to optionally separate out your docs into many files...just save them to the `includes` folder and add them to the top of your `index.md`'s frontmatter. Files are included in the order listed.

    The Kittn API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request sucks.
    401 Unauthorized -- Your API key is wrong.
    403 Forbidden -- The kitten requested is hidden for administrators only.
    404 Not Found -- The specified kitten could not be found.
    405 Method Not Allowed -- You tried to access a kitten with an invalid method.
    406 Not Acceptable -- You requested a format that isn't json.
    410 Gone -- The kitten requested has been removed from our servers.
    418 I'm a teapot.
    429 Too Many Requests -- You're requesting too many kittens! Slow down!
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
